# mmann1123.github.io

Personal academic website for Michael L. Mann — Associate Professor, Department of Geography & Environment, The George Washington University.

Live at: https://mmann1123.github.io/

## Structure
- `index.html` — single-page site
- `colors_and_type.css` — design tokens (colors, typography)
- `assets/` — portrait, logos, icons

## Local preview
Any static server works, e.g.:
```bash
python -m http.server 8000
```
Then visit http://localhost:8000/
